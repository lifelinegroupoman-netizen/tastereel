# Restaurant Portal
Next.js + Supabase + Cloudinary + Stripe scaffold for a restaurant-focused video/review platform.

Steps:
1. Copy files into a new Next.js project (or unzip this repo).
2. Run `npm install`.
3. Create a Supabase project and run SQL from `/migrations/init.sql`.
4. Fill `.env.local` with your keys.
5. Start dev server: `npm run dev`.
